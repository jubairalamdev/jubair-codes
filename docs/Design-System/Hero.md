# Hero Section Design Specification

> Purpose: Define the visual structure and design language of the Hero section.

---

# Overall Layout

The Hero section occupies the larger left portion of the first screen.

Characteristics:

- Large rounded container
- Floating card appearance
- Spacious layout
- Strong typography hierarchy
- Minimal visual clutter
- Comfortable whitespace throughout

The content is vertically centered inside the container.

---

# Container

The hero is enclosed inside a large card.

Properties:

- Large border radius
- Thin subtle border
- Soft shadow
- White background
- Floating appearance
- Generous internal padding

---

# Background

The background is intentionally minimal.

Characteristics:

- Very subtle radial glow
- Uses the Accent Color with low opacity
- No patterns
- No illustrations
- No heavy gradients

The glow should never distract from the content.

---

# Content Hierarchy

Content flows vertically in this order:

1. Status Badge
2. Main Heading
3. Description
4. Primary Actions
5. Social Links

Everything is left aligned.

---

# Status Badge

A small pill badge displayed above the heading.

Characteristics:

- Small rounded pill
- Thin border
- Accent Color indicator
- Minimal typography
- Compact horizontal padding

Purpose:

Display current availability or current activity.

---

# Heading

The heading is the dominant element.

Characteristics:

- Very large font size
- Bold font weight
- Black text
- Tight line height
- Maximum visual emphasis

This should be the first thing users notice.

---

# Description

Positioned directly below the heading.

Characteristics:

- Medium font size
- Secondary text color
- Comfortable line height
- Limited text width
- High readability

---

# Action Buttons

Three horizontally aligned buttons.

Structure:

Primary Button

- Filled style
- Rounded pill
- Uses Primary Color
- White text

Secondary Button

- Outlined or soft filled style
- Uses Accent Color
- Includes icon
- Rounded pill

Tertiary Button

- Minimal outlined button
- Rounded pill
- Includes icon

Spacing between buttons should be consistent.

---

# Social Section

Located near the bottom of the card.

Structure:

- Small horizontal container
- Label
- Circular icon buttons

Characteristics:

- Rounded container
- Thin border
- Small icon buttons
- Equal spacing
- Minimal appearance

---

# Typography

Hierarchy:

- Large bold heading
- Medium description
- Small badge text
- Medium button text
- Small social label

---

# Spacing

The Hero relies heavily on whitespace.

Guidelines:

- Large internal padding
- Comfortable spacing between every content block
- Buttons grouped closely
- Plenty of breathing room around the heading

---

# Borders

Use:

- Thin subtle borders
- Large rounded corners
- Consistent border radius across buttons and badges

---

# Shadows

Use only soft elevation.

No heavy shadows.

---

# Design Principles

The Hero should communicate:

- Confidence
- Professionalism
- Simplicity
- Modern design
- Premium quality

Avoid:

- Decorative graphics
- Large illustrations
- Excessive animations
- Visual clutter
