# Contact Section Design Specification

> Purpose: Define the overall layout and design language of the Contact section.

---

# Overall Layout

The Contact section is presented as a large floating container divided into two columns.

Characteristics:

- Large rounded container
- Floating appearance
- Spacious layout
- Two-column structure
- Clear content separation
- Comfortable whitespace

Desktop Layout

```
---------------------------------------------------------
|                                                       |
| Contact Information | Contact Form                    |
|                     |                                 |
|                     |                                 |
---------------------------------------------------------
```

Tablet

- Single-column layout
- Contact information appears first
- Contact form appears below

Mobile

- Fully stacked layout
- Full-width content

---

# Container

Properties:

- Large border radius
- Thin subtle border
- White background
- Soft shadow
- Spacious internal padding

---

# Left Column

The left side introduces communication methods.

Content Order

1. Section Header
2. Introduction Text
3. Contact Methods
4. Social Links

---

# Section Header

Structure

- Decorative indicator
- Section title
- Divider line

Example

Let's Connect

---

# Introduction

Short paragraph introducing communication.

Example Placeholder

I'm always open to discussing new opportunities, freelance projects, collaborations, or simply having a conversation about web development.

Characteristics

- Comfortable line height
- Secondary text
- Limited width

---

# Contact Methods

Displayed as reusable contact cards.

Characteristics

- Full width
- Rounded container
- Thin border
- Horizontal layout
- Comfortable padding
- Small icon
- Primary information
- Optional action icon

Example Placeholders

Email

hello@example.com

Phone

+8801XXXXXXXXX

---

# Social Links

Displayed beneath the contact methods.

Characteristics

- Circular icon buttons
- Equal spacing
- Consistent sizing
- Minimal appearance

Example Platforms

- LinkedIn
- GitHub
- Facebook
- Discord
- YouTube

---

# Right Column

Contains the contact form.

The form itself is displayed inside a reusable card component.

---

# Responsive Behaviour

Desktop

Two-column layout.

Tablet

Stack vertically.

Mobile

Single-column layout with full-width elements.

---

# Typography

Hierarchy

- Section Title
- Form Title
- Labels
- Input Text
- Supporting Text
- Button Text

---

# Spacing

Maintain generous spacing between:

- Header
- Introduction
- Contact methods
- Social links
- Form card

---

# Borders

Use

- Thin subtle borders
- Rounded containers
- Rounded contact cards

---

# Shadows

Use subtle elevation for:

- Outer container
- Form card

Internal components remain flat.

---

# Design Principles

The Contact section should communicate:

- Accessibility
- Professionalism
- Simplicity
- Trust
- Clean communication

Avoid

- Large illustrations
- Decorative graphics
- Visual clutter
