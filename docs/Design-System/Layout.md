# Layout System

---

# Maximum Width

Every section shares the same container width.

```
max-width: 1200px
```

All content should remain centered horizontally.

---

# Container

Desktop

1200px maximum width.

Tablet

Fluid width with horizontal padding.

Mobile

100% width with horizontal padding.

---

# Section Spacing

Every section should maintain generous vertical spacing.

Consistency is more important than exact values.

---

# Grid System

Desktop

Multi-column layouts.

Tablet

Reduced columns.

Mobile

Single-column layouts.

---

# Cards

Every major section is presented inside floating cards.

Maintain:

- Equal padding
- Equal spacing
- Equal border radius

---

# Responsive Rules

Desktop First

↓

Tablet

↓

Mobile

Maintain consistent spacing across every breakpoint.
