# Motion & Interaction

---

# Philosophy

The portfolio intentionally avoids large entrance animations.

Performance and clarity take priority.

---

# Loading

Use pulsing skeleton loaders.

Never use spinner loaders unless absolutely necessary.

---

# Hover

Allowed

- Button hover
- Card hover
- Link hover
- Navigation hover

Hover interactions should be subtle.

---

# Page Loading

Use Next.js loading UI.

Loading states should match the final layout.

---

# Transitions

Keep transitions short and smooth.

Avoid exaggerated animations.

---

# Scrolling

Use smooth scrolling for navigation.

No scroll reveal animations.

---

# Focus

Interactive components should clearly indicate keyboard focus.

---

# Rules

No Framer Motion.

No GSAP.

No decorative animations.

Motion should improve usability only.
