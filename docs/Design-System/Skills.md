# Skills Section Design Specification

> Purpose: Define the visual structure and design language of the Skills section.

---

# Overall Layout

The Skills section is displayed as a floating information card divided into multiple categories.

Characteristics:

- Large rounded container
- Floating appearance
- Well organized
- Easy to scan
- Minimal design

---

# Container

Properties:

- Large border radius
- Thin subtle border
- White background
- Soft shadow
- Comfortable internal padding

---

# Section Header

Structure:

- Decorative indicator
- Section title
- Divider line

Example:

Title:
Skills

---

# Skill Categories

Skills are grouped into categories.

Each category contains multiple skill badges.

Example Categories:

Core Stack

Additional Technologies

Tools

---

# Category Heading

Small heading displayed above each skill group.

Characteristics:

- Medium font weight
- Uppercase or small title
- Comfortable spacing

Example Placeholder:

Core Stack

---

# Skill Badge

Each skill is displayed inside a reusable pill component.

Characteristics:

- Rounded pill
- Thin border
- Small icon
- Small typography
- Equal spacing
- Consistent height

Example Skills

HTML

CSS

JavaScript

TypeScript

React

Next.js

Node.js

Express.js

MongoDB

Tailwind CSS

Git

GitHub

---

# Certifications

A separate content group displayed below the skills.

Each certification is shown inside its own rounded container.

Characteristics:

- Thin border
- Small icon
- Comfortable padding
- Full width

Example Certifications

Frontend Development Certificate

Responsive Web Design

JavaScript Algorithms

---

# Layout Flow

Section Header

↓

Skill Categories

↓

Skill Badges

↓

Certifications

---

# Typography

Hierarchy:

- Section Title
- Category Heading
- Badge Text
- Certification Text

---

# Spacing

Maintain generous spacing between:

- Categories
- Skill rows
- Certification cards

Badges should wrap naturally onto new lines.

---

# Borders

Use:

- Thin subtle borders
- Rounded badges
- Rounded certification cards

---

# Shadows

Only the outer card uses elevation.

Internal badges remain flat.

---

# Design Principles

The Skills section should communicate:

- Technical expertise
- Organization
- Simplicity
- Professionalism

Avoid:

- Progress bars
- Percentage indicators
- Complex charts
- Visual clutter
