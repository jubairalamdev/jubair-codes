# Experience Section Design Specification

> Purpose: Define the visual structure and design language of the Experience section.

---

# Overall Layout

The Experience section is presented as a large floating card that displays a vertical timeline of professional experiences.

Characteristics:

- Large rounded container
- Floating appearance
- Structured layout
- Clean typography
- Comfortable whitespace
- Multiple experience entries

---

# Container

Properties:

- Large border radius
- Thin subtle border
- White background
- Soft shadow
- Spacious internal padding

---

# Section Header

Structure:

- Decorative indicator
- Section title
- Divider line

Example:

Title:
Experience

---

# Experience Item

Each experience is displayed inside its own content block.

Structure:

1. Job Title
2. Date
3. Company Tags
4. Description
5. Expand Link

Items are stacked vertically.

Separate every experience with a subtle divider.

---

# Job Header

Layout:

Left:

- Job Title

Right:

- Employment Date

Example Placeholder:

Job Title:
Frontend Developer

Employment Date:
January 2025 — Present

---

# Company Tags

Small pill badges positioned below the job title.

Characteristics:

- Rounded pills
- Thin border
- Small typography
- Equal spacing

Example Placeholders:

Company:
Your Company

Employment Type:
Remote

Location:
Dhaka, Bangladesh

---

# Description

A short paragraph describing the role.

Characteristics:

- Comfortable line height
- Secondary text
- Limited width
- Easy readability

Example Placeholder:

Worked on modern web applications, collaborated with cross-functional teams, and developed scalable frontend solutions.

---

# Expand Link

Located below each description.

Characteristics:

- Small typography
- Inline icon
- Minimal appearance

Example Placeholder:

View Details

---

---

# Typography

Hierarchy:

- Medium-Large Job Title
- Medium Date
- Small Tags
- Medium Description
- Small Action Link

---

# Spacing

Maintain consistent spacing between:

- Job Header
- Tags
- Description
- Action Link

Each experience block should have generous vertical spacing.

---

# Borders

Use:

- Thin subtle borders
- Rounded badges
- Subtle divider between experiences

---

# Shadows

Only the outer container uses elevation.

Internal content remains flat.

---

# Design Principles

The Experience section should communicate:

- Professionalism
- Organization
- Readability
- Simplicity

Avoid:

- Timeline graphics
- Heavy decorations
- Large icons
- Visual clutter
