# About Section Design Specification

> Purpose: Define the visual language and layout of the About card displayed beside the Hero section.

---

# Overall Layout

The About section is presented as a compact information card.

Characteristics:

- Tall card
- Rounded container
- Floating appearance
- Structured content
- Comfortable spacing
- Minimal design

The layout is vertically stacked.

---

# Container

Properties:

- Large border radius
- Thin subtle border
- White background
- Soft shadow
- Floating appearance
- Consistent padding

---

# Section Header

Located at the top.

Structure:

- Small decorative indicator
- Section title
- Divider line

Characteristics:

- Horizontal layout
- Medium font weight
- Clean spacing
- Minimal decoration

---

# Profile Image

Centered horizontally below the section header.

Characteristics:

- Circular image
- Soft shadow
- Thin border
- Medium size

Purpose:

Acts as the primary visual element of the card.

---

# Description

Appears directly below the image.

Characteristics:

- Multiple short paragraphs
- Comfortable line height
- Secondary text color
- Medium font size
- Left aligned

Paragraph spacing should improve readability.

---

# Information Badge

Positioned near the bottom.

Characteristics:

- Rounded pill
- Thin border
- Small icon
- Small typography
- Compact padding

Purpose:

Display a single piece of important information.

---

# Layout Flow

Content order:

1. Section Header
2. Profile Image
3. Description
4. Information Badge

Everything follows a clean vertical rhythm.

---

# Typography

Hierarchy:

- Medium section title
- Medium body text
- Small badge text

Maintain consistent spacing between each level.

---

# Spacing

Guidelines:

- Large internal padding
- Equal spacing between sections
- Comfortable paragraph spacing
- Plenty of breathing room around the image

---

# Borders

Use:

- Thin subtle borders
- Rounded corners
- Consistent radius across badges

---

# Shadows

Use:

- Soft elevation
- Subtle depth
- No harsh shadows

---

# Design Principles

The About card should communicate:

- Trust
- Professionalism
- Simplicity
- Personality
- Premium quality

Avoid:

- Large decorative elements
- Complex layouts
- Heavy borders
- Visual clutter
