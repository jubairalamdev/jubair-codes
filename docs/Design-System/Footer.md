# Footer Design Specification

> Purpose: Define the visual structure and layout of the website footer.

---

# Overall Layout

The footer is intentionally minimal.

It serves as the closing section of the portfolio without distracting from the main content.

Characteristics

- Center aligned
- Lightweight
- Minimal typography
- Comfortable spacing

---

# Layout

Structure

```
----------------------------
© Year Your Name
All Rights Reserved.
----------------------------
```

Everything remains horizontally centered.

---

# Content

Example Placeholder

© 2026 Your Name

All rights reserved.

---

# Typography

Characteristics

- Small font size
- Secondary text
- Medium readability
- Center aligned

---

# Spacing

Provide generous spacing between:

- Contact section
- Footer

The footer should have comfortable top and bottom padding.

---

# Borders

No border required.

The footer relies on whitespace for separation.

---

# Shadows

No shadows.

---

# Responsive Behaviour

Desktop

Centered.

Tablet

Centered.

Mobile

Centered.

No layout changes required.

---

# Design Principles

The footer should communicate:

- Simplicity
- Professionalism
- Completion

Avoid

- Large navigation
- Decorative graphics
- Heavy styling
- Visual distractions
