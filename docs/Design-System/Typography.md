# Typography

---

# Font Family

Use a modern sans-serif font across the entire application.

Maintain consistency.

---

# Typography Scale

Hero Heading

- Largest typography

Section Heading

- Large typography

Card Heading

- Medium-large typography

Body

- Comfortable reading size

Small

- Supporting information

Caption

- Labels
- Badges

Button

- Medium weight
- Easy readability

---

# Font Weight

Light

Normal

Medium

Semi Bold

Bold

Use only these weights.

---

# Line Height

Paragraphs should have comfortable reading spacing.

Large headings should remain compact.

---

# Text Alignment

Default

Left aligned.

Exceptions

- Footer
- Certain badges

---

# Rules

Never use more than one font family.

Typography hierarchy should always remain consistent.
