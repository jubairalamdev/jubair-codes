# Navbar Design Specification

> **Purpose:** Define the visual and structural design of the portfolio website's navigation bar. This document serves as a reference during development to ensure consistency with the chosen design inspiration.

---

## Design Style

The navigation bar follows a **minimal**, **premium**, and **modern** design language similar to high-end SaaS websites.

Characteristics:

- Clean and uncluttered
- Large amount of whitespace
- Soft shadows for subtle depth
- Rounded (pill-shaped) container
- Neutral color palette
- Elegant typography
- No unnecessary borders or decorations

---

The navbar consists of **three sections**.

```
--------------------------------------------------------------
| Logo |           Navigation Links           | Download CV   |
--------------------------------------------------------------
```

### Right

Replace the theme toggle from the inspiration with:

**Download CV Button**

---

# Navbar Container

The container should have:

- Pill-shaped border radius
- White (or off-white) background
- Thin subtle border
- Soft box shadow
- Floating appearance
- Center aligned
- Large horizontal padding
- Comfortable vertical padding

Overall feeling:

> A floating navigation bar that feels lightweight and premium.

---

# Typography

Style:

- Modern sans-serif font
- Medium font weight
- Neutral gray text color
- Clear visual hierarchy
- High readability

---

# Spacing

The design relies heavily on whitespace.

Guidelines:

- Large horizontal padding
- Even spacing between navigation items
- Balanced alignment between left, center, and right sections
- Comfortable height without feeling oversized

---

# Shadows & Depth

Use only subtle elevation.

Characteristics:

- Soft shadow
- No harsh contrast
- Creates the illusion of floating above the page

---

# Borders

- Thin, subtle border
- Border should barely be noticeable
- Helps define the navbar without making it feel heavy

---

# Visual Principles

The navbar should communicate:

- Simplicity
- Professionalism
- Premium quality
- Modern web design
- Excellent readability

Avoid:

- Heavy gradients
- Thick borders
- Bright colors
- Large icons
- Unnecessary animations
- Visual clutter

---
